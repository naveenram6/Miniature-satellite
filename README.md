# Multi-Sensor Satellite Monitoring System

## 📌 Overview
This project simulates a miniature satellite system capable of monitoring environmental and motion parameters in real time. It integrates multiple sensors and displays live data while also storing readings using an SD card for analysis.

## 🎯 Objective
To design a compact embedded system that performs real-time data acquisition, monitoring, and storage similar to a satellite telemetry system.

## ⚙️ Parameters Measured
- 🌡️ Temperature (Climate Monitoring)  
- 🌫️ Air Quality (MQ2 Gas Sensor – Smoke detection)  
- 🔄 Motion & Orientation (MPU6050 – Gyroscope X, Y, Z)  
- 💾 Data Logging (SD Card Storage)  

## ⚙️ Working Principle
Sensors continuously collect environmental and motion data:

- Temperature sensor measures ambient conditions  
- MQ2 sensor detects smoke/gas levels  
- MPU6050 provides gyroscope data (X, Y, Z axes)  

The microcontroller processes this data and:
- Displays it in real time (UI/serial interface)  
- Stores it in an SD card for future analysis  

## 🛠️ Components Used
- MQ2 Gas Sensor  
- MPU6050 (Gyroscope + Accelerometer)  
- Temperature Sensor  
- SD Card Module  
- Microcontroller (Raspberry Pi Pico / Arduino)  
- Breadboard & Power Supply  

## 🚀 Features
- Real-time data monitoring  
- Multi-sensor integration  
- SD card data logging  
- Satellite-like telemetry system  
- User interface for visualization  

## 🔧 Technologies Used
- Embedded Systems  
- I2C Communication (MPU6050)  
- Analog/Digital Sensor Integration  
- Data Logging Systems
